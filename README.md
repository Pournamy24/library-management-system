# Library Management System

## Project Overview

The Library Management System is a console-based application developed using C++. It is designed to manage library books efficiently using Object-Oriented Programming (OOP) concepts and file handling. The system allows users to add books, search books, issue books, return books, and display all available books.

## Objectives

* To develop a menu-driven Library Management System.
* To implement Object-Oriented Programming concepts using classes.
* To use file handling for permanent storage of book records.

## Features

* Add New Book
* Display All Books
* Search Book by Title
* Search Book by Author
* Issue Book
* Return Book
* Store book records using file handling

## Technologies Used

* GCC Compiler (MSYS2)

## OOP Concepts Used

* Class
* Objects


## Expected Output

The application provides a menu-driven interface to perform various library operations such as adding books, displaying books, searching by title or author, issuing books, and returning books while maintaining persistent data using file handling.
